# login-form-with-animation

Animated login form concept with glassmorphism styling and simple interaction states.

Language: **English** | [Русский](README.ru.md)

## Problem
- Need a clean, reusable login UI concept for portfolio demos and landing pages.
- Need a lightweight implementation without framework/runtime dependencies.

## Stack
- HTML5
- CSS3
- Ionicons (CDN)

## Setup
```bash
git clone https://github.com/xelvhk/login-form-with-animation
cd login-form-with-animation
open index.html
```

Alternative (Python local server):
```bash
python3 -m http.server 8080
# open http://localhost:8080
```

## Architecture
- `index.html`: semantic login form structure.
- `style.css`: animation, layout, and visual styles.
- `example*.jpg`: static visual previews.
