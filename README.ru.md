# login-form-with-animation

Концепт анимированной формы входа в стиле glassmorphism с простыми интерактивными состояниями.

Язык: [English](README.md) | **Русский**

## Проблема
- Нужен чистый и переиспользуемый UI-концепт логина для портфолио и лендингов.
- Нужна легковесная реализация без фреймворков и runtime-зависимостей.

## Стек
- HTML5
- CSS3
- Ionicons (CDN)

## Setup
```bash
git clone https://github.com/xelvhk/login-form-with-animation
cd login-form-with-animation
open index.html
```

Альтернатива (локальный сервер на Python):
```bash
python3 -m http.server 8080
# открыть http://localhost:8080
```

## Архитектура
- `index.html`: семантическая структура формы входа.
- `style.css`: анимация, layout и визуальные стили.
- `example*.jpg`: статические превью.
